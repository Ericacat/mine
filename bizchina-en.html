BizCHINA
Education Platform

I redesigned an online platform for learning business Chinese.
I handled both design and Vue.js coding. The result was a fresh new look and a more efficient system.

Role
Product Designer
UI/UX + Frontend
Period
Feb 2024 – Present
Company
Netstar Inc.
Team Size
6 people (small team)
Challenge

Challenges

1. The Old Design Did Not Look Modern

The old site looked outdated. It did not match what users expect from a modern online school. This hurt the brand and made it hard to get new users.

2. The Old System Was Hard to Update

The old system cost a lot of time and money to update. Every small change took too much work. This was a big problem for the team.

3. Too Many Outside Tools

The team used outside tools for classes and teacher communication. It was hard to track attendance or manage lessons well. It also cost more money.

Solution 1

Brand Refresh & UI/UX Improvement

We updated the whole visual design and improved the user experience.

1-1. New Visual Design

What we did:

  • Updated all brand colors and fonts
  • Built a modern and friendly design system
  • Made the whole site look consistent

Results:

  • The site now looks modern
  • The design is consistent across all pages
  • The design system makes it easy to add new features

1-2. Simpler Sign-Up Flow

Problem

After the redesign, more users signed up for a free trial. But not many became paying members. The 5-step sign-up process was too long and stopped users from finishing.

What Each Person Wanted

  • Management: Show the price clearly at the start
  • Japanese staff: Explain all the details to users
  • Users: Start quickly and easily

My Approach

I listened to each person's needs and found a solution that worked for everyone:

  • Reduced from 5 steps to 2 steps
  • Show the price first (for management)
  • Add details in the next steps (less pressure for users)
  • Changed from free trial to a low-cost paid trial (to attract serious users)

Results

✅ Users could start learning faster. More users became paying members.
✅ Customer support questions dropped a lot.

Solution 2

Frontend Rebuild & System Update

We had a 6-person team with no frontend engineer. So I did both design and coding.

Why This Worked Well

  • No gap between design and the final result
  • No need to check the design back and forth
  • The result matched the design exactly
  • Development was faster

Case 1: Responsive Design

Problem

The old site had two separate versions: one for desktop and one for mobile. This took twice as much work to maintain.

Solution

I built a responsive design with Vue.js. One code works for all devices. This saved a lot of time and money.

Results

  • Lower development cost (no more duplicate code)
  • Easier to maintain (one codebase)
  • Better experience on mobile

Case 2: Web Classroom System

Problem

The team used outside video tools. It was hard to check who came to class or how lessons were going.

Solution

  • I designed and built our own web classroom
  • I connected the video system to an easy-to-use interface
  • Users had a smooth experience from start to finish

Results

  • Less need for outside tools
  • All lessons are now managed inside the platform
  • The team can check class status in real time
Solution 3

Instructor WeChat Mini Program

Problem

Teachers had to use many different tools just to check their schedule. This was slow and confusing.

Solution

  • I designed and built a WeChat Mini Program for teachers using Vue.js + uni-app
  • Teachers can check schedules, materials, and student progress on their phone
  • Course data is shown in a clear and easy-to-read format

Results

✅ Teachers have less work to do
✅ Teachers said: "It's so easy to manage on my phone"
✅ Class preparation became faster

Results & Impact

Results & Learnings

Numbers

  • Customer support questions dropped a lot
  • It became faster to add new features after the rebuild
  • Responsive design removed the need to maintain two separate versions

Other Results

  • The brand now looks modern and professional
  • Users can start learning more quickly
  • Teachers said the new tools are easier to use
  • The team works more efficiently

What I Learned

  • In a small team, it helps a lot when the designer can also write code
  • When one person does both design and coding, the result matches the plan exactly
  • By improving step by step, we could make big changes without stopping the service

Platform Expansion

We are still adding new features. We also made a new platform called NetKOREA for learning Korean. We used the same system but made a new design for it. This showed the system can work for other languages too.
