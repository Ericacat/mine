BizCHINA
Education Platform

I redesigned an online platform for learning business Chinese.
I handled both design and Vue.js implementation — so the experience I envisioned was the experience users received.

Role
Product Designer
UI/UX + Frontend
Period
Feb 2024 – Present
Company
Netstar Inc.
Team Size

Challenge

Challenges

1. The Old Design Did Not Look Modern

The old site looked outdated. It did not match what users expect from a modern online school. This hurt the brand and made it hard to get new users.

2. An Outdated System That Stood in the Way

The existing system made every small change a significant effort. It became a barrier to improving the experience over time.

3. A Fragmented Experience from Outside Tools

Relying on outside tools for classes and communication fragmented the learning flow. There was no way to offer a seamless, end-to-end experience within the platform.

Solution 1

Brand Refresh & Experience Redesign

We renewed the visual language and rethought the experience from the ground up.

1-1. New Visual Design

What we did:

  • Updated all brand colors and fonts
  • Built a modern and friendly design system
  • Made the whole site look consistent

Results:

  • The site feels fresh and welcoming
  • Every page reads as part of the same experience
  • The design system supports new features without breaking consistency

1-2. Simpler Sign-Up Flow

Problem

After the redesign, interest grew — but many users didn't complete sign-up. The 5-step process created too much friction before users could even get started.

What Each Person Wanted

  • Management: Show the price clearly at the start
  • Japanese staff: Explain all the details to users
  • Users: Start quickly and easily

My Approach

I listened to each person's needs and found a solution that worked for everyone:

  • Reduced from 5 steps to 2 steps
  • Show the price first (for management)
  • Add details in the next steps (less pressure for users)
  • Changed from free trial to a low-cost paid trial (to attract serious users)

Results

✅ Users could start learning without hesitation. More completed the journey to become paying members.
✅ Fewer users reached out for help — because they didn't need to.

Solution 2

Frontend Rebuild & System Update

We had a 6-person team with no frontend engineer. So I did both design and coding.

Why This Worked Well

  • What I designed is what users experience — no translation loss
  • Subtle details survive all the way to the final build
  • The experience stays in my hands from concept to delivery
  • From decision to working product, the path is short

Case 1: Responsive Design

Problem

The old site had separate desktop and mobile versions. The experience could differ between them — and keeping both in sync was a constant challenge.

Solution

I built a responsive design with Vue.js — one codebase, one consistent experience across every device.

Results

  • A consistent experience on every device
  • One codebase to maintain
  • Mobile users feel just as at home

Case 2: Web Classroom System

Problem

Outside video tools broke the continuity of the learning experience. There was no way to bring lessons and learner data together in one place.

Solution

  • I designed and built our own web classroom
  • Video delivery and an intuitive interface — connected as one
  • The learning journey now flows without interruption

Results

  • The learning journey flows without interruption
  • All lessons are now managed inside the platform
  • The team can check class status in real time
Solution 3

Instructor WeChat Mini Program

Problem

Teachers had to jump between multiple tools just to prepare for a single class. The constant switching broke their focus and flow.

Solution

  • I designed and built a WeChat Mini Program for teachers using Vue.js + uni-app
  • Teachers can check schedules, materials, and student progress on their phone
  • Course data is shown in a clear and easy-to-read format

Results

✅ Everything a teacher needs lives in one place
✅ Teachers said: "It's so easy to manage on my phone"
✅ Teachers can focus on teaching, not on navigating tools

Results & Impact

Results & Learnings

Numbers

  • Fewer users reached out for help — because they didn't need to
  • New features can be built without disrupting the existing experience
  • One codebase serves every device with a consistent feel

Other Results

  • The brand now feels modern and inviting
  • Users can begin learning without hesitation
  • Teachers said the new tools are easier to use
  • The team can respond to what users need, faster

What I Learned

  • In a small team, it helps a lot when the designer can also write code
  • When one person owns design and implementation, the experience survives intact to the end
  • A step-by-step approach let us transform the experience without disrupting the users who rely on it

Platform Expansion

We are still adding new features. We also made a new platform called NetKOREA for learning Korean. We used the same system but made a new design for it. This showed the system can work for other languages too.
